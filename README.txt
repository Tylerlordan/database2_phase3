To Install and Run:
-Run Apache and MySQL server on XAMPP
-Place the PHP files in the htdocs folder inside of XAMPP
-Create a database named 'db2'
-Run the given sql file inside of the database
-Run Android Studio
-In app\src\main\res\values\strings.xml change the url value to your ipv4 address 
along with the folder name where the php files are located, ex. http://9999.9999.9999/PHPandSQL/
-Create an android device emulator with API version 33 to run the app
-Run the app in Android Studio
